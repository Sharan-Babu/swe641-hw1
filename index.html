Projects

KirinEdit
  • Teaching Large Language Models (LLMs) to 'edit' text input by identifying and replacing text sequences in it, without rewriting the entire text.
  • Significantly decreases latency and cost as you would only pay for substitute tokens.
  • Opens up use-cases that require quick feedback like Long Document Editing and Coding.
UserSubContext
  • Novel context management technique for Large Language Models (LLMs).
  • Uses 'current user message' to dynamically select only a subset of the current context window for next turn based on previous user messages.
  • Reduces token consumption and cost. Better approach than naïve rolling window based methods.
